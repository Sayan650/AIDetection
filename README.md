# Disease Detection App

A React Native application for disease detection through image analysis.

## Setup Complete

The project has been initialized with the following dependencies:

### Core Dependencies
- React Native (Expo)
- TypeScript
- React Navigation v6
- React Native Paper
- NativeWind (Tailwind CSS)
- React Native Reanimated
- React Native Image Picker
- AsyncStorage

### Project Structure
```
src/
├── components/
│   └── common/
├── screens/
├── navigation/
├── services/
├── utils/
└── constants/
```

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Run on specific platform:
   ```bash
   npm run android  # Android
   npm run ios      # iOS (requires macOS)
   ```
